GitHub Repo

About

Illegal Aliens is one of the projects that are closest to my heart. It's not the best or most beautiful creation, but it's arguably the project that taught me the most. It was carried out in the course object-oriented programming project. As a first year student, it was our first major project, building upon our previous courses in programming.

The requirement was to build an application using the MVC-architecture (Model-View-Controller). We carried out an entire process, spanning from researching frameworks to implementation using UML-diagrams. As the base language, Java was used. To handle dependencies, we used Gradle and we used the framework LibGDX to built the user interface and underlying game logic. During the process, we created both a System Design Document, as well as a Requirement and Analysis Document.

Arguably, the most important learning was to properly plan out the project structure (folders and classes) to avoid circular dependencies in the code. As we were approaching the deadline, we analyzed the project architecture and found a mind-boggling amount of circular dependencies in our code. We spent 12 hours in a single day constantly rewriting and restructuring the project to avoid the circular dependencies. Something that could have easily been avoided with proper initial planning.

In the end, we were very happy about the results. We had, for our first time, carried out an entire implementation project and achieved the highest grade for it.
